## 项目介绍

本项目使用three.js的包装库kokomi.js，并使用vue+vite+ts进行工程化开发，镜头动画使用gsap库实现

粒子使用着色器材质(使用GPU渲染)，大幅度提升了性能，不过目前还存在不少问题(自己对着色器不是很懂)，粒子动画还有些生硬，后期有时间再进行优化。
